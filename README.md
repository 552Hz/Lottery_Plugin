# 地下偶像抽签 AstrBot 插件

QQ 群里 `@机器人 /抽签` 后，返回一张地下偶像主题的每日运势图。

## 功能

- 每个 QQ 号每天的结果固定，以上海时区零点刷新。
- 运势随机为 `大吉`、`吉`、`中`、`凶`、`大凶`。
- 从 `吃ota饭`、`反切`、`关门`、`看活`、`规划远征`、`睡觉`、`喝酒`、`版聊` 中抽取 4 个，随机分到 `宜` 和 `忌`。
- 返回极简 PCIE 科技风图片，底图由代码生成，不依赖外部图片素材。

## 安装

把本目录放到 AstrBot 的 `data/plugins/` 下，确保插件目录里有：

- `main.py`
- `fortune_card.py`
- `metadata.yaml`
- `requirements.txt`
- `README.md`
- `.gitignore`
- `__init__.py`

AstrBot 会按 `requirements.txt` 安装 `Pillow`。如果部署环境缺少中文字体，建议安装 Noto CJK 字体，否则图片中文字可能显示为方框。

## 使用

在群聊里发送：

```text
@机器人 /抽签
```

插件会生成并缓存当天图片，同一 QQ 号当天重复抽签会得到同一张图。
